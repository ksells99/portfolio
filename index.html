About Me

I am a self-taught front-end developer, skilled in a number of technologies and frameworks including React and Angular. With a professional background in software support, I am mindful of developing user-friendly, responsive applications that can be targeted to a wide audience.

I have also become familiar with a few back-end technologies whilst developing a number of full-stack applications, such as MongoDB, Firebase, Node.js and Express.

A passion for self-improvement means I am always keen to learn the latest industry-relevant technologies and implement these into my applications.

Outside of work and web development, you'll either find me spending time with my family, tinkering with the latest tech, or watching a borderline obsessive amount of Formula 1 and other motorsport on TV.

F1Latest

F1Latest shows the latest Formula 1 information, including a countdown to the next race, season calendar and the current standings, with data obtained from the Ergast API. It also includes a dark mode switch, which is persisted to the user's local storage.

HTML, CSS, Bootstrap, TypeScript, Angular, Ergast API
Live Site View on Github

recip-ez

recip-ez is a MERN stack application which allows users to browse for recipes (fetched from the Spoonacular API) and save them to their own collection. State management is handled by Redux, and the app features full authentication.

HTML, CSS, Bootstrap, JavaScript, React, Redux, MongoDB, Express, Node.js, Spoonacular API
Live Site View on Github

MyCarDealership

MyCarDealership is an Angular stock management portal designed for car dealerships, allowing them to keep track of vehicle stock and view a summary via an intuitive dashboard. Dealerships can sign up for a free account and add/manage their stock collection. The application is powered by Firebase.

HTML, SCSS, Bootstrap, TypeScript, Angular, Firebase/Firestore, Chart.js, Google Maps API, LocationIQ API
Live Site View on Github

Session Booker

Session Booker allows users to create 'sessions' and add/manage bookings against them (e.g. breakfast or after school clubs). It is a MERN stack application featuring full user authentication, and uses Redux for state management.

HTML, CSS, Bootstrap, JavaScript, React, Redux, MongoDB, Express, Node.js
Live Site View on Github
